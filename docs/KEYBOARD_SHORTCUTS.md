# CareConnect — Keyboard Shortcut Sheet

Every primary action is accessible without a mouse.
Shortcuts follow the standard desktop conventions and are available
from the native menu bar, the contextual toolbar, and through direct key presses.

On macOS, use **⌘ (Cmd)** wherever **Ctrl** is listed.

## Global

| Shortcut | Action | Where |
|----------|--------|-------|
| `Ctrl + N` | Create a new record / appointment | Dashboard |
| `Ctrl + S` | Save / submit the current form | Profile, Settings |
| `Ctrl + F` | Focus the search bar (stays on the current screen) | Global |
| `Enter` | Run the search (from the search bar) | Search bar |
| `Ctrl + ,` | Open Settings | Global |
| `F1` | Open the keyboard shortcuts help dialog | Global |
| `Esc` | Close dialog / cancel / clear the search bar | Global |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + 1` | Go to Dashboard |
| `Ctrl + 2` | Go to Notifications |
| `Ctrl + 3` | Go to Profile |
| `Tab` / `Shift + Tab` | Move forward / backward between panels and controls |
| `Arrow keys` | Move within the menu bar, toolbar, sidebar, and tab lists (roving focus) |
| `Home` / `End` | Jump to first / last item in a toolbar or tab list |

## Accessibility (apply immediately)

| Shortcut | Action |
|----------|--------|
| `Ctrl + Alt + H` | Toggle High Contrast Mode (WCAG AAA palette) |
| `Ctrl + Alt + L` | Toggle Left-Handed Layout (mirror the interface) |
| `Ctrl + +` | Increase text scale (up to 2.0×) |
| `Ctrl + -` | Decrease text scale (down to 0.8×) |
| `Ctrl + 0` | Reset text scale to 1.0× |
| `Ctrl + \` | Show / hide the right info panel |

## Focus order

On the login screen, focus runs from the email field to the password field, then to the
show or hide button, the forgot-password link, the sign-in button, and the high-contrast
switch. When a sign-in fails, focus moves to the error and then to the first field that
needs fixing.

In the app, Tab moves from the skip link to the menu bar, then to the toolbar, the search
box, the sidebar, the main content, and finally the right panel. A dialog keeps focus
inside itself while it is open. Esc closes the dialog and returns focus to whatever opened
it. When the screen changes, focus moves to the new heading so a screen reader reads the
new context.

## Notes

The shortcut hints show up in three places. They appear in the keyboard list in the right
panel, in the Keyboard tab under Settings, and in the F1 help dialog. Ctrl+N and Ctrl+S
also sit on the File menu, so people can find them without memorizing anything.
